# WEBPLAYER MAXGO

### Requisitos del Sistema

El siguiente software es necesario para ejecutar WebPlayer Max Pro:

- **Sistema Operativo**: Ubuntu 20.04.6 LTS (Focal Fossa)
- **Aplicación**: XUI ONE
